    Types of images

    • PNG - Portable Network Graphic
      • Example sample image
    • SVG - Scalable Vector Graphic
      • Example star image
    • GIF - Graphics
      • Allows image animation
      • Only supports 256 color
      • Does not support transparency
      • Used with line art cartoon characters
      • Small file size but can be terribly quality for photos
    • JPG - Joint Photographic Experts Group
      • Supports 16.7 Million colors
      • Offers file compression
      • Best for photos and controlling the quality
